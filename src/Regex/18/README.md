# Challenge 18 - Matching same text again & again

You have a test string $`S`$.

Your task is to write a regex that will match $`S`$ with the following conditions:

- $`S`$ must be of length: `20`
- $`1^{st}`$ character: `lowercase letter`.
- $`2^{st}`$ character: `word character`.
- $`3^{st}`$ character: `whitespace character`.
- $`4^{st}`$ character: `non word character`.
- $`5^{st}`$ character: `digit`.
- $`6^{st}`$ character: `non digit`.
- $`7^{st}`$ character: `uppercase letter`.
- $`8^{st}`$ character: `letter` (either lowercase or uppercase).
- $`9^{st}`$ character: `vowel` (a, e, i , o , u, A, E, I, O or U).
- $`10^{st}`$ character: `non whitespace character`.
- $`11^{st}`$ character: should be same as `1st character`.
- $`12^{st}`$ character: should be same as `2nd character`.
- $`13^{st}`$ character: should be same as `3rd character`.
- $`14^{st}`$ character: should be same as `4th character`.
- $`15^{st}`$ character: should be same as `5th character`.
- $`16^{st}`$ character: should be same as `6th character`.
- $`17^{st}`$ character: should be same as `7th character`.
- $`18^{st}`$ character: should be same as `8th character`.
- $`19^{st}`$ character: should be same as `9th character`.
- $`20^{st}`$ character: should be same as `10th character`.
